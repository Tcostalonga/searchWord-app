# searchWordApp
 
 
## Introdução

**Aplicativo para pesquisa online do significado de palavras em inglês.**

### Pré-requisitos

Para instalar o app: 
- Smarthphone: Mínimo Android 6.0 - Marshmallow - API 23


Para executar o código: 
- Android Studio: mín 4.0
- Gradle: mín 4.0
